## Filebrowser

```bash
curl -fsSL https://raw.githubusercontent.com/filebrowser/get/master/get.sh | bash
filebrowser -a 0.0.0.0 -r /

```

## Ativar Filebrowser

filebrowser -a 0.0.0.0 -r /

## Ativar homelab outra porta

```bash
curl -fsSL https://raw.githubusercontent.com/filebrowser/get/master/get.sh | bash
filebrowser -a 0.0.0.0 -p 8085 -r /
```
